👑 King-Void-MDX

A powerful Multi-Device WhatsApp bot built on the Baileys library.

✨ Features

- Multi-Device WhatsApp support
- AI-powered commands
- Downloader commands
- Group management
- Fun commands
- Utility commands
- Media tools
- Easy deployment
- Plugin support
- Fast and lightweight

🚀 Installation

git clone https://github.com/K-Void-mdx/King-Void-MDX.git
cd King-Void-MDX

npm install

cp config.env.example config.env

npm start

⚙️ Configuration

Edit "config.env" and set your required variables before starting the bot.
